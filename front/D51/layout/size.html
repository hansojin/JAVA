이 div 요소는 max-width 속성을 500px로 설정했습니다.

설정된 너비 이하로 브라우저의 크기가 줄어들때 해당 요소에 스크롤바를 생성하게 됩니다.
이 div 요소는 min-width 속성을 500px로 설정했습니다.

min-width 속성이 설정된 요소에 width 속성값을 따로 설정하지 않으면 초기 너비(width)는 100%를 가지게 되고, 웹 브라우저의 크기가 줄어들면 거기에 맞춰서 HTML 요소의 너비가 자동으로 줄어듭니다.
하지만 min-width속성으로 설정된 너비 이하로 줄어들면, HTML 요소의 너비는 더 이상 줄어들지 않고, 수평 스크롤 바를 생성하게 됩니다.

웹 브라우저의 크기를 줄여서 두 div 요소의 차이를 알아보세요!
