선언 위치에 따라
1. 멤버 변수 : '클래스 영역'에 선언
2. 지역 변수 : '메소드나 생성자 내부'에 선언


1. 멤버 변수는 다시 "클래스 변수와 인스턴스 변수" 로 나눌 수 있음
	1_1. 인스턴스 변수
	 : 각각의 개체(인스턴스) 마다 개별적인 속성을 가져야 할 때 
	 (ex. 게임에서 캐릭터 머리색, 옷색 정하는 거 처럼)
	1_2. 클래스 변수 
	 : 모든 객체가 공통적으로 똑같은 속성을 가질때 사용
	 (ex. 모든 캐릭터가 2m의 키를 갖도록 고정시키는 등)
	 : 일반적으로 앞에 static이 붙어있으면 클래스 변수 (전역함수의 성질을 가짐)

2. 지역 변수
 : 메소드 내에 선언되어 메소드 내에서만 사용이 가능한 변수로, 메소드 종료시 자동으로 소멸됨
