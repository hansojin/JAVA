== 멤버변수 기본값== 
자료형(변수 타입)	기본값
byte		0
short		0
int		0
long		0L
float		0.0f
double		0.0d
char		'\u0000'
boolean		false
참조형 변수(String or any Object)	null


* 참조형 변수 : 주소 값을 저장하는 변수

간단히 정리하자면 기본형 변수는 Stack영역에서 실제 값을 그대로 저장하지만
참조형 변수는 먼저 Stack영역에 공간을 생성하고 Heap영역에 새로운 공간을 생성해
실제 값을 저장한 다음에 그 공간의 주소 값을 불러와서 Stack영역의 공간에 저장한다는 것


예를 들어, int i=10은, 
stack 영역에 선언한 변수 만큼의 공간이 생성되고 그 안에 값이 들어감. 
즉, stack 영역에 4byte 크기의 i라는 공간이 생기고, 그 안에 10이 들어감

String str = new String("java")의 경우에는, 
기본형 변수처럼, stack 영역에 str이라는 4byte 크기의 공간이 생성됨. 
여기에는 0x100 의 주소값이 저장이 되고, 
new 라는 명령어가 heap 영역에 새로운 저장 공간을 생성함
0x100이라는 주소값 공간(=메모리 주소값)은 우리가 넣으려는 값인 "java" 크기 만큼 생성됨. 

즉, 참조형 변수는 기본형 변수와 다르게 실제 값을 그대로 저장하는 공간이 아니라, 
Heap 영역에 값이 저장된 공간의 주소 값을 저장하는 공간임. 
그래서 참조형 변수는 주소 값을 저장하는 변수임
